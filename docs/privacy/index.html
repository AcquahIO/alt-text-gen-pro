Alt Text Generator Pro - Privacy Policy

Last updated: February 3, 2026

This Privacy Policy explains how the Alt Text Generator Pro Chrome extension ("we", "us", or "our") handles information when you use the extension.

Information We Collect

  • Images and image URLs. When you right-click an image or upload files, the extension may process the image or its URL to generate alt text.
  • Page context. To improve alt text quality, the extension can read nearby page context such as page title, nearby headings, surrounding text, aria labels, and data-* attributes.
  • User-provided context. Optional text you enter to guide generation.
  • Account and subscription data. If you sign in, the extension stores an auth token, user ID, email, display name, and avatar URL to manage access and subscription status.
  • Local usage data. The extension stores recent alt text results locally (alt text, image URL, page title, timestamp) and pending uploads to support the UI.
  • API key and settings. If you provide your own API key or endpoint, these values are stored in Chrome sync storage so they can be used across devices.

How We Use Information

  • Generate alt text for images you select or upload.
  • Provide context-aware descriptions that better match the page content.
  • Maintain your session and subscription status if you sign in.
  • Show recent results and pending uploads inside the extension UI.

Where Data Is Processed

Image and context data are sent to the endpoint configured in the extension. By default, this may be our hosted backend. If you configure your own endpoint, data is sent to that endpoint instead. If you provide an OpenAI API key, data may be sent directly to OpenAI for processing. Third-party services have their own privacy policies and terms.

Data Retention

  • Local storage: recent results, pending uploads, and settings remain on your device until you clear them or uninstall the extension.
  • Server processing: images and context are used to generate alt text. We do not intentionally store image content beyond processing, but operational logs or error traces may be retained for reliability and abuse prevention.

Your Choices

  • Clear recent results from the extension UI.
  • Remove your API key or endpoint in settings.
  • Sign out to clear stored auth data.
  • Uninstall the extension to remove all local data.

Security

We use standard security practices to protect data in transit. No method of transmission is 100% secure, and we cannot guarantee absolute security.

Children's Privacy

This extension is not intended for children under 13, and we do not knowingly collect personal information from children.

Changes to This Policy

We may update this policy from time to time. The "Last updated" date above reflects the most recent revision.
